Implementation notes: The neural network used was trained using the MNIST dataset (link). Preprocessing includes bounding box, center of mass, and converting into bins. Model was trained using Python and weights were exported to web interface written in Javascript. Net architecture: input layer of 728 (28x28 pixels) hidden layer 100, output layer 10 with softmax activation function. Early stopping to avoid overfitting
